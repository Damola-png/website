# Adedamola Olayefun — Personal Website (Static)

This is a **plain HTML/CSS/JS** portfolio site built to deploy on **Netlify**.

## Quick edit checklist
- Update project descriptions/links in `script.js`
- Replace placeholder resumes:
  - `assets/resume-cs.pdf`
  - `assets/resume-media.pdf`
- Optional: Add 6–12 real photos into `assets/photos/` and update the `photos` list in `script.js`

## Deploy to Netlify (fast)
Netlify → **Add new site** → **Deploy manually** (Drag & Drop) → drop the folder (or zip).

### Contact form
This uses **Netlify Forms**. After deployment:
Netlify Dashboard → Site → **Forms** → view submissions there.
